# Chapter 27: Final Project

## Learning Objectives

- Students will be able to collect and integrate data from multiple sources including numeric, categorical, text, and image data for a comprehensive analytics project
- Students will be able to apply the complete CRISP-DM pipeline to a real- world predictive analytics problem from problem framing through evaluation
- Students will be able to combine multiple modeling techniques and present findings with actionable recommendations to stakeholders
